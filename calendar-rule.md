# 日历事件展示规则

## 规则概述

### 1. 连续日期区间规则
- 如果事件是连续日期区间（如1月15-18日），**只在起始日期显示一次**
- 事件标题格式：`类别:事件名称`
- 日期范围通过标题括号中的数字表示，如：`Q4(15-18)`

### 2. 月度重复事件规则
- 每月固定日期的多日事件，**只在每月第一天显示**
- 例如：每月10-12日的WASDE报告，只在每月10号显示一次

### 3. 周期性事件规则
- **每周四/每周一**的事件：每个周四/周一都显示
- 不进行合并处理

## 实现说明

### 代码实现要点

1. **事件类型定义**
   ```typescript
   type CalendarCategory = 'soybean' | 'pig' | 'legend' | 'lithium';
   ```

2. **连续区间检测**
   - 对于相同标题的事件，检测是否为连续日期
   - 如果连续，只保留第一个日期的事件
   - 在标题中添加日期范围标注

3. **动态日期生成**
   - 周四/周一事件：使用JavaScript Date对象动态生成全年所有符合条件日期

4. **数据结构**
   ```typescript
   interface CalendarEvent {
     date: string;  // MM-DD格式
     title: string;
     description: string;
     category: CalendarCategory;
   }
   ```

## 维护指南

### 添加新事件类别

1. 在 `CalendarEvent` 的 `category` 类型中添加新类别
2. 在 `getCategoryColor` 和 `getCategoryBorderColor` 函数中添加对应颜色
3. 在图例区域添加对应的圆点和说明

### 添加新事件

1. 在 `parseCalendarEvents` 函数中添加事件对象
2. 确保日期格式为 `MM-DD`
3. 设置正确的 category 类型
4. 按照上述规则设置是否需要显示日期范围

### 修改现有事件

- **修改标题**：直接更新 `title` 字段
- **修改日期**：更新 `date` 字段（格式：月-日）
- **修改描述**：更新 `description` 字段
- **修改类别**：更新 `category` 字段
