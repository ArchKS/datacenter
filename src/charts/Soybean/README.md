# 豆粕市场供需数据

## 数据来源
- USDA（美国农业部）供需报告
- Mysteel农产品数据
- 饲料行业信息网

## 更新频率
季度更新

## 数据说明
- **美豆产量**: 美国大豆产量（单位：亿蒲式耳）
- **美豆库存**: 美国大豆期末库存（单位：亿蒲式耳）
- **巴西产量**: 巴西大豆产量（单位：亿吨）
- **阿根廷产量**: 阿根廷大豆产量（单位：亿吨）
- **中国油厂库存**: 中国油厂大豆库存（单位：万吨）

## 图表类型
- **多系列折线图**: 展示美豆产量、库存和南美产量趋势
- **双Y轴**: 左侧显示美豆数据，右侧显示南美和中国数据

## 市场特点
- 全球大豆供需整体宽松
- 南美（巴西、阿根廷）丰产预期强烈
- 中国油厂库存处于历史高位
- 美豆出口创13年新低
