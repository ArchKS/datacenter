# 锂矿库存分析 (Lithium Inventory Analysis)

## 图表概述

本模块展示锂矿行业的周度库存与产能数据，帮助分析锂市场供需动态。

## 数据来源

- **数据来源**:
  - 本地历史数据（2024年9月-12月）
  - SMM上海有色网周报数据
  - 东方财富网碳酸锂库存报告
  - 富宝资讯、Mysteel等第三方数据平台
- **更新频率**: 每周四更新
- **时间范围**: 2024年9月 - 2026年1月
- **数据单位**: 吨
- **数据完整性说明**:
  - 2024年9月-10月：完整的库存结构和产能数据
  - 2024年12月-2026年1月：主要为总库存数据，部分周度包含库存结构数据
  - 产能数据：仅2024年10月31日-12月12日有详细数据
  - 缺失数据标记为 `null`

## 图表说明

### 1. 总库存趋势
- **图表类型**: 折线图 + 面积填充
- **Y轴**: 库存量（吨）
- **说明**: 展示锂矿总库存的周度变化趋势，面积填充突出库存规模
- **颜色**: 绿色渐变 (#22C55E)

### 2. 库存结构
- **图表类型**: 堆叠柱状图 + 双Y轴
- **左Y轴**: 库存量（吨）
- **右Y轴**: 环比增长率（%）
- **数据系列**:
  - 上游冶炼厂库存（绿色）
  - 下游企业库存（浅绿）
  - 其他库存（淡绿）
- **增长率曲线**:
  - 上游增长率（红色线）
  - 下游增长率（橙色线）
  - 其他增长率（蓝色线）
- **说明**: 分析不同环节库存分布及周度环比变化

### 3. 周度产能
- **图表类型**: 分组柱状图 + 双Y轴
- **左Y轴**: 产能（吨/周）
- **右Y轴**: 环比增长率（%）
- **数据系列**:
  - 锂辉石产量（绿色）
  - 锂云母产量（浅绿）
  - 盐湖产量（淡绿）
  - 回收产量（浅绿）
- **增长率曲线**: 各来源产能周度环比变化
- **说明**: 对比不同锂来源的周度产能及增长趋势

### 4. 去库量对比
- **图表类型**: 柱状图
- **Y轴**: 去库量（吨）
- **说明**: 展示每周库存净减少量，反映去库存速度
- **颜色**: 绿色渐变
- **注意**: 第一周无去库数据（无基准对比）

### 5. 产能总量
- **图表类型**: 柱状图
- **Y轴**: 产能（吨/周）
- **说明**: 展示每周总产能规模及环比变化
- **颜色**: 绿色渐变

## 技术实现

- **框架**: React + TypeScript
- **图表库**: ECharts 5.5.0
- **组件**: `LithiumChart.tsx`
- **数据文件**: `lithium_data.json`

### 数据结构

```typescript
interface LithiumDataPoint {
  date: string;              // MM-DD格式
  totalInventory: number;    // 总库存
  inventory: {
    upstream: number;        // 上游冶炼厂库存
    downstream: number;      // 下游企业库存
    other: number;           // 其他库存
  };
  destocking: number | null; // 去库量
  totalProduction: number | null;      // 总产能
  production: {
    lithiumSpodumene: number | null;   // 锂辉石产量
    lithiumLepidolite: number | null;  // 锂云母产量
    saltLake: number | null;           // 盐湖产量
    recycling: number | null;          // 回收产量
  };
}
```

## 维护指南

### 更新数据

1. 编辑 `src/charts/Lithium/data/lithium_data.json`
2. 按照现有格式添加新的周度数据
3. 日期格式：`MM-DD`（如 `12-19`）
4. 首周数据：
   - `destocking` 设为 `null`（无基准对比）
   - `production` 相关字段设为 `null`（产能数据从第7周开始）

### 添加新数据周

```json
{
  "date": "12-19",
  "totalInventory": 110000,
  "inventory": {
    "upstream": 18000,
    "downstream": 42000,
    "other": 50000
  },
  "destocking": 1469,
  "totalProduction": 22000,
  "production": {
    "lithiumSpodumene": 13800,
    "lithiumLepidolite": 2900,
    "saltLake": 3000,
    "recycling": 2300
  }
}
```

## 注意事项

1. **增长率计算**: 系统自动基于相邻两周数据计算环比增长率
2. **产能数据缺失**: 前5周（09-18至10-24）产能数据为null，周度产能图表仅显示有数据的周
3. **去库数据**: 首周（09-18）无去库数据，去库量对比图表从第二周开始显示
4. **颜色一致性**: 所有子图表使用绿色系（#22C55E及其变体），与日历中"锂矿"类别的绿色标识保持一致

## 关键指标

### 2024年数据（9月-12月）
- **总库存趋势**: 从13.75万吨（09-18）降至11.15万吨（12-12），累计去库2.6万吨（19%）
- **库存结构**: 上游库存持续下降，下游库存先升后降，其他库存相对稳定
- **产能水平**: 周度产能稳定在2.1-2.2万吨
- **去库速度**: 平均周去库2300吨，峰值在11-14周（3481吨）

### 2026年1月数据（来自SMM）
- **1月29日**: 总库存10.75万吨，环比去库1414吨
  - 冶炼厂：1.90万吨（去库831吨）
  - 下游材料/电池：4.06万吨（补库3007吨）
  - 贸易商：4.79万吨（去库3590吨）
- **1月累计去库**: 2123吨（超预期，市场原预期累库8000-10000吨）
- **连续去库**: 已连续8周去库，累计去库约5337吨（降幅4.74%）
- **库存天数**: 约25-28.5天，处于历史低位

### 2026年2月数据（来自SMM/富宝）
- **2月5日**: 总库存10.52万吨，周度产量20744吨（环比减3.8%）
- **2月12日**: 总库存10.29万吨，环比去库2531吨，库存天数31.7天
  - 下游材料厂占比升至43.2%（补库节奏持续）
- **2月19日**: 总库存约10.12万吨，估算去库1500吨
- **2月26日**: 总库存9.90万吨，环比去库3170吨
  - 冶炼厂：1.8万吨（历史低位）
  - 下游：4.3万吨
  - 其他：3.8万吨
- **2月累计去库**: 约6000吨，延续快速去库趋势
- **库存趋势**: "盐厂低库存、下游高库存"格局

### 2026年3月数据
- **状态**: 3月库存数据尚未发布（SMM数据通常每周四更新）
